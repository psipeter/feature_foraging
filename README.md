# 2.5D Foraging Experiment
**Psychology Experiment developed in Unity**

## Overview
This project uses a 2.5D perspective to study stimulus-action-goal mappings. 
Participants navigate a 3D environment with constrained movement.

## Tech Stack
* **Engine:** Unity 2022.3 LTS (or your version)
* **Scripting:** C#
* **OS:** Pop_OS 22.04

## Setup
1. Clone the repository.
2. Open with Unity Hub.
3. Ensure Git LFS is installed for assets.
